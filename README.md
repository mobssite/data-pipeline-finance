# End-to-End Data Pipeline & Business Analytics Project

## Objective
Design and implement an end-to-end data pipeline to analyze sales performance and support business decision-making.

## Architecture
CSV → Python → SQLite → SQL → Power BI

## Tools
- Python (Pandas)
- SQLite
- SQL
- Power BI

## Key Features
- Data ingestion, cleaning, and transformation
- Data modeling for business reporting
- KPI calculation (Revenue, Top Customers, Top Products)
- Interactive dashboard development

## Business Insights
- Identified strong monthly revenue fluctuations and seasonal trends
- Highlighted top-performing customers contributing significantly to total revenue
- Identified key products driving business performance
- Provided clear KPIs to support strategic decision-making

## Results
- Total revenue successfully calculated from raw transactional data
- Monthly trends extracted using SQL queries
- Top 10 customers and products identified

### Example Metrics
- November 2018 Revenue: 117,938
- December 2018 Revenue: 82,825
- Top Customer: SM-20320 → 25,043
- Top Product: TEC-CO-10004722 → 61,599

## Business Impact
This project demonstrates the ability to transform raw data into actionable insights, enabling better understanding of revenue drivers and supporting data-driven decision-making.
