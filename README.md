# End-to-End Data Pipeline & Business Analytics Project

## Objective
Build an end-to-end data pipeline to analyze financial performance.

## Architecture
CSV → Python → SQLite → SQL → Power BI

## Tools
- Python (Pandas)
- SQLite
- SQL
- Power BI

## Key Features
- Data ingestion and cleaning
- Data modeling
- KPI calculation
- Dashboard visualization

## Business Insights
- Monthly revenue trends
- Top performing stores
- Sales patterns analysis

## Results

- Total Revenue calculated from transactional data
- Monthly revenue trends identified using SQL
- Top 10 customers and products analyzed
